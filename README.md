## Deployment

### Connect tomcat

1. Download tomcat core->zip.
1. Unzip it to java folder.
1. Project structure -> libraries -> add -> tomcat/lib/servlet-api.jar.
1. Add web.xml.
1. Add tomcat in running configuration.

### Add libraries

The all storing in lib folder.
